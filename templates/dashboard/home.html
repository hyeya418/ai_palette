AI Palette

Generative AI와 RAG 기반의 혁신적인 문서, 요약, 챗봇 솔루션

주요 기능

완벽 요약

  • 유튜브 요약: Speech-to-Text AI를 이용한 동영상 스크립트와 요약 제공
  • 문서 요약: PDF, DOCX, XLSX, TXT 등 다양한 포맷 지원
  • 웹사이트 요약: URL 기반의 실시간 요약
  • 텍스트 요약: 입력된 텍스트를 분석하고 간결한 요약 제공

PPT 초안

  • 업로드한 파일을 벡터화하여 RAG 기술 활용
  • 등록한 URL 기반으로 구글 검색 API를 활용한 콘텐츠 생성

레포트 생성

다양한 형식의 데이터를 기반으로 전문 레포트 초안을 생성

발표 대본

입력된 자료를 바탕으로 발표에 최적화된 대본 작성

면접 준비

  • 채용공고 URL 및 이력서 PDF 업로드
  • 예상 면접 질문 자동 생성

문서 기반 챗봇

PDF, DOCX, XLSX, TXT 기반의 문서 내용으로 AI와 실시간 대화
